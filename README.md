# University Learning Management System (ULMS)

## Project Description

University Learning Management System is a web-based application developed to manage university learning activities.
This system helps students to register, login, view courses, download lecture notes, access assignments, and manage their profiles.
Administrators can manage students, courses, lecture notes, and assignments.

## Features

### Student Module

- Student Registration
- Student Login
- Student Dashboard
- View Courses
- Download Lecture Notes
- View Assignments
- Edit Profile
- Logout

### Admin Module

- Admin Dashboard
- View Students
- Manage Courses
- Add Courses
- Upload Lecture Notes
- Upload Assignments
